# Project Context

## Project

This is a frontend capstone project built with React and Vite.

## Technology Stack

- React
- Vite
- JavaScript
- CSS
- React Router

## Coding Conventions

- Use functional React components.
- Use PascalCase for component names and filenames.
- Use camelCase for variables and functions.
- Keep components small and focused.
- Prefer readable code over unnecessary abstraction.
- Use semantic HTML and accessible attributes.
- Avoid adding dependencies unless necessary.
- Explain significant changes before applying them.
- Do not modify unrelated files.

## Git Conventions

Use Conventional Commits:

- `feat:` for new features
- `fix:` for bug fixes
- `docs:` for documentation
- `refactor:` for code restructuring
- `style:` for formatting-only changes
- `chore:` for tooling and maintenance

Keep commit descriptions concise and imperative.

## Commands

- Install dependencies: `npm install`
- Start development server: `npm run dev`
- Run linting: `npm run lint`
- Build project: `npm run build`
