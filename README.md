# FlyRank Capstone

A frontend capstone project built with React and Vite.

## Tech Stack

- **Framework:** [React](https://react.dev/)
- **Build Tool:** [Vite](https://vite.dev/)
- **Linting:** [ESLint](https://eslint.org/)

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v18 or higher recommended) and npm installed on your machine.

### Installation

1. Navigate to the project directory.
2. Install the project dependencies:
   ```bash
   npm install
   ```

### Available Scripts

In the project directory, you can run the following commands:

- **`npm run dev`**: Starts the local development server at `http://localhost:5173`.
- **`npm run build`**: Compiles and optimizes the application into static files for production inside the `dist` folder.
- **`npm run preview`**: Serves the local production build to preview it before deploying.
- **`npm run lint`**: Runs ESLint to analyze code quality and detect style or syntax issues.
